# Lite-XML
A very lightweight XML construction library

[![npm version](https://badge.fury.io/js/lite-xml.svg)](https://badge.fury.io/js/lite-xml)

## Installation

You can install lite-xml using npm, yarn, or pnpm:

```bash
npm install lite-xml
```

```bash
yarn add lite-xml
```

```bash
pnpm add lite-xml
```
